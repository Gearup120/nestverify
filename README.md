# NestVerify 🏠

**NestVerify** is a premium, AI-powered real estate platform designed to solve the trust gap in the Nigerian rental market. It features verified property listings, secure payment integration, and an intelligent AI assistant.

## 🚀 Features
- **Verified Listings:** Every property is physically inspected and verified.
- **NestAI Assistant:** An AI-powered chatbot that recommends properties directly from the database.
- **Cinematic Hero Slider:** A high-end visual experience for home seekers.
- **Landlord Dashboard:** Comprehensive tools for managing listings and viewing requests.
- **Smooth Navigation:** Interactive UI with smooth scrolling and responsive design.

## 🛠️ Tech Stack
- **Frontend:** React, Vite, Tailwind CSS, Lucide Icons.
- **Backend:** Django, Django REST Framework, SQLite (Development).
- **AI:** Groq (Llama 3.1) for intelligent property recommendations.
- **Images:** Cloudinary for high-performance image hosting.

## 📦 Installation & Setup

### Backend
1. `cd house_back`
2. `python -m venv venv`
3. `source venv/bin/activate` (or `venv\Scripts\activate` on Windows)
4. `pip install -r requirements.txt`
5. `python manage.py migrate`
6. `python manage.py runserver`

### Frontend
1. `cd housing`
2. `npm install`
3. `npm run dev`
